
# 🚀 Machine Learning Optimization using Particle Swarm Optimization (PSO)

## 📌 Project Overview

This project implements a Machine Learning workflow integrated with **Particle Swarm Optimization (PSO)** to optimize model performance. The notebook performs data preprocessing, feature scaling, model training, and evaluation using Python's machine learning ecosystem.

The project is designed to work with datasets stored in **Excel (.xlsx)** format and is executed in **Google Colab**.

---

## 🎯 Objectives

* Upload and process real-world datasets from Excel files.
* Perform data preprocessing and feature scaling.
* Split the dataset into training and testing sets.
* Build and train a Machine Learning model using TensorFlow/Keras.
* Apply Particle Swarm Optimization (PSO) for parameter optimization.
* Evaluate and improve model performance.

## ▶️ How to Run

1. Open `ML-code.ipynb` in Jupyter Notebook or Google Colab.
2. Upload your Excel dataset (`.xlsx`) when prompted.
3. Ensure the dataset contains:

   * Feature columns
   * A target column named `target`
4. Execute all notebook cells sequentially.
5. View the optimized model results and performance metrics.
